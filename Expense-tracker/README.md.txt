# Expense Tracker (Desktop Application)

A beginner-friendly personal expense tracker built using Python.

## Features
- User-wise expense tracking
- Monthly budget management
- Calendar-based date selection
- Expense visualization using charts
- Data stored using SQLite

## Technologies Used
- Python
- Tkinter
- SQLite
- Matplotlib
- tkcalendar

## How to Run
1. Install required packages:
   pip install -r requirements.txt
2. Run the program:
   python main.py

## Purpose of This Project
This project was created as a learning exercise to practice:
- Python programming
- GUI development
- Database handling
- Debugging and improving application flow

## Future Improvements
- Add logout and user switching
- Export expenses to CSV or PDF
- Improve UI design
- Create a web-based version in the future
